# Guia de Uso - Integração MontSystem

Este guia apresenta as instruções para utilizar a integração entre o site da Monteiro Solução Contábil e o sistema MontSystem.

## Acessando o MontSystem

1. No menu principal do site, clique na opção **MontSystem**
2. Você será direcionado para a página de apresentação do sistema
3. Clique no botão **Acessar Sistema**
4. No modal de login, insira suas credenciais:
   - **Email:** admin@contabilmonteiro.com.br
   - **Senha:** L1b3rd4d3@
5. Clique em **Entrar**
6. Você será redirecionado para o sistema MontSystem

## Criando uma Nova Conta

1. No menu principal do site, clique na opção **MontSystem**
2. Clique no botão **Acessar Sistema**
3. No modal de login, clique na aba **Criar Conta**
4. Preencha os campos solicitados:
   - Nome
   - Sobrenome
   - Email
5. Clique em **Criar Conta Grátis**
6. Você receberá uma confirmação e poderá fazer login com as credenciais criadas

## Navegando pelo Site

A integração permite navegar facilmente entre o site institucional e o sistema MontSystem:

- O menu principal contém a opção **MontSystem** em todas as páginas
- A página de apresentação do MontSystem contém informações detalhadas sobre o sistema
- O botão **Conhecer Funcionalidades** leva à seção de funcionalidades na página de apresentação
- O botão **Fale Conosco** direciona para a página de contato do site
